# Kris Kringle Online

Welcome to the Kris Kringle Online project! This repository contains the code for a web application that helps groups organize and manage their Kris Kringle (Secret Santa) gift exchange online. Whether you're planning a holiday gift exchange for family, friends, or coworkers, this project aims to make the process easy and fun.

## Features
- **Easy Group Creation:** Set up a Kris Kringle group with just a few clicks.
- **Participant Management:** Add or invite participants easily.
- **Automated Pairing:** Randomly assign Kris Kringle pairs, ensuring everyone gets a unique partner.
- **Wish Lists:** Participants can create and share wish lists to help their Kris Kringle choose the perfect gift.
- **Email Notifications:** Automatic notifications to participants about their assigned partners and important updates.
- **Secure and Private:** All data is handled securely to ensure privacy.
